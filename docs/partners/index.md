# Rue 合作伙伴

Rue 致力于与技术服务商、云与基础设施供应商、数据与 AI 平台、开发者工具与集成方、教育与社区组织建立长期合作关系，共同提升前端开发体验与生态繁荣。

## 合作类型

- 解决方案与实施：提供基于 Rue 的咨询、交付与培训
- 技术与基础设施：云、边缘、浏览器/运行时、监控与性能平台
- 插件与集成：路由、状态管理、UI 组件、构建与测试工具
- 教育与内容：课程、文档翻译、社区活动与技术传播
- 渠道与推广：联合市场活动、案例共创与媒体合作

## 合作收益

- 官方“合作伙伴”标识与认证
- 官网与文档中展示案例与链接
- 技术支持与优先沟通渠道
- 联合发布、直播与线下活动
- 行业解决方案与参考架构共建

## 如何加入

- 在 GitHub 提交合作意向 Issue（hunzhiwange/ruejs 仓库）：说明公司介绍、能力范围、典型案例、预期合作方向
- 我们将进行评估并在 7 个工作日内与您联系推进

## 现有合作与更新

我们会持续更新合作伙伴名单与案例展示，敬请关注。
