# 成为 Rue 赞助商

Rue 是一个采用 MIT 许可证的开源项目，完全免费使用。项目的持续维护、生态建设与功能迭代需要投入大量精力与资源。我们衷心感谢赞助商的支持，正是你们的贡献让 Rue 得以健康、稳定地发展。

## 如何赞助 {#how-to-sponsor}

目前，请通过 GitHub Issues 提交您的赞助意向：
https://github.com/hunzhiwange/ruejs/issues

请在提交时说明：

- 公司/个人简介
- 意向赞助类型（一次性或按月循环）
- 意向赞助周期（如适用）
- 预计赞助金额

我们支持**一次性赞助**和**按月循环赞助**。选择循环赞助可依据赞助等级享受相应的标识与展示权益。如有需要，我们可以提供合规的发票或相关材料。

若您对赞助等级、支付方式或曝光数据有任何疑问，请直接在 Issues 中留言，我们将尽快回复。

## 企业赞助 Rue

赞助 Rue 能为您的企业带来面向全球前端开发者的持续曝光（通过官网和 GitHub 项目展示）。这不仅有助于吸引潜在客户，更能塑造企业“积极拥抱和支持开源”的正面品牌形象。

- **依赖 Rue？** 赞助 Rue 的开发能确保这个您所依赖的基础项目保持健康、活跃并持续维护。同时，在开发者社区的曝光能提升您团队的招聘效率和品牌传播力。
- **目标客户是开发者？** Rue 社区的曝光为您带来高质量的目标流量，有效提升品牌认知度和潜在转化率。

## 个人赞助 Rue

如果您是个人用户，并且认可 Rue 为您带来的效率提升，欢迎进行小额赞助，就像“请我们喝杯咖啡”一样简单而温暖的支持。

同时，我们也非常鼓励您向所在公司推荐 Rue，推动他们以企业身份进行赞助——这通常能为开源项目提供更稳定、长久的支持。
