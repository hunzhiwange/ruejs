# 翻译 {#translations}

## 可用语言 {#available-languages}

- [简体中文 / 简体中文](https://cn.ruejs.org/) [[source](https://github.com/rue-js-translations/docs-zh-cn)]

## 开始新的翻译 {#starting-a-new-translation}

Rue 文档最近进行了重大修订，因此其他语言的翻译仍然缺失或正在进行中。

我们欢迎社区努力提供更多翻译。翻译工作由 [rue-js-translations](https://github.com/rue-js-translations/) GitHub 组织管理。如果您有兴趣贡献，请查看[翻译指南](https://github.com/rue-js-translations/guidelines/blob/main/README.md)开始。
