# Rue.js 隐私政策

> 生效日期：2025年1月3日

本隐私政策旨在说明Rue.js组织（以下简称"我们"）在运营本网站及相关开源服务网站（统称"网站"）以及提供相关内容、文档、信息和服务（如教程、开发者工具、资源获取等）过程中处理用户信息的原则与实践。

需特别说明的是，本政策不适用于以下情形：

1. GitHub平台（）上开源代码、文档或规范的使用，此类行为受相应开源许可协议约束；
2. 参与GitHub开源项目的交互行为（如提交PR、问题反馈等），适用GitHub平台条款；
3. NPM平台（）上我们发布的软件包使用统计，受NPM服务条款规制；
4. 浏览器/IDE厂商收集的我们发布的扩展程序使用数据，遵循各厂商自身条款。

## 信息收集范围

我们郑重声明：**不通过任何渠道（包括网站、NPM软件包或浏览器/IDE扩展）收集或存储个人数据**。网站仅通过第三方服务商获取匿名化数据：

- **网站访问统计**：采用符合GDPR、ePrivacy、COPPA及CCPA规范的[Fathom Analytics]进行隐私友好型数据分析。该服务仅临时处理IP地址且无法识别个人身份，符合CCPA去标识化要求（详见Fathom隐私政策：legal/privacy）
- **搜索功能数据**：基于[Algolia DocSearch]实现，该服务禁用用户追踪与指纹识别技术，不使用Cookie，已获得GDPR、CCPA合规认证及TRUSTe认证（详见Algolia隐私政策：/policies/privacy 与安全合规说明：）

## 数据使用目的

所有数据收集仅服务于唯一目的：以最大限度保护隐私的方式分析网站流量与使用模式，从而持续优化网站体验与文档质量。依据GDPR规定，其法律基础为"第6(1)(f)条：基于持续改进网站与业务的合法利益"。需要强调的是，所有数据处理均不涉及长期存储个人身份信息。

## 数据留存机制

所有匿名数据均存储于上述第三方服务平台，其保存期限完全遵循各服务商的数据留存政策。
