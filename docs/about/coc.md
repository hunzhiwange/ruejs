# 行为准则

## 我们的承诺

为营造开放包容的社区环境，我们作为项目贡献者和维护者郑重承诺：无论年龄、体型、残障状况、民族、性征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族，每位参与者都将在项目和社区中获得免受骚扰的参与体验。

## 行为准则

有助于营造积极环境的行为包括：

- 使用友善包容的语言表达
- 尊重不同的观点与经历
- 以优雅姿态接受建设性批评
- 聚焦于对社区最有利的决策
- 展现对社区成员的同理心

不可接受的行为示例如下：

- 恶意挑衅、侮辱性或贬损性言论，以及人身攻击
- 公开或私下的骚扰行为
- 未经明确授权公开他人隐私信息（如物理或电子地址）
- 其他在专业场合中被合理认定为不当的行为

## 责任说明

项目维护者有责任明确可接受行为的标准，并应在出现不可接受行为时采取适当且公正的纠正措施。维护者有权并有责任删除、编辑或拒绝不符合本行为准则的评论、代码提交、维基编辑、议题等贡献内容，并可对存在不当、威胁、冒犯或有害行为的贡献者采取临时或永久禁言措施。

## 适用范围

本行为准则适用于所有项目空间，同时当个人在公共空间代表项目或社区时也同样适用。代表项目或社区的情形包括使用官方项目邮箱、通过官方社交媒体账号发言，或作为指定代表参与线上/线下活动等。项目维护者可对"代表行为"进行进一步界定与说明。

## 执行机制

如遭遇侮辱、骚扰或其他不可接受行为，可通过ruejs@qq.com联系项目团队。所有投诉都将经过审核调查，并根据实际情况采取必要且适当的应对措施。项目团队有义务对举报人信息严格保密。具体执行政策的详细说明可能另行发布。

未真诚遵循或执行本行为准则的项目维护者，可能会根据项目领导层其他成员的决定面临临时或永久性的后果。
