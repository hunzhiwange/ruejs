# 10：迁移 Transition 与 TransitionGroup

## 目标

处理 built-ins 中最依赖区间语义和 DOM 时序的两项能力。

## 只做这些事

1. Transition 直接消费 Renderable / Block。
2. TransitionGroup 直接消费 Renderable / Block。
3. 保持 enter、leave、appear、FLIP 行为稳定。

## 建议涉及文件

1. packages/runtime/src/components/Transition.ts
2. packages/runtime/src/components/TransitionGroup.ts
3. 必要的动画测试文件

## 不要顺手做

1. 不改 Router。
2. 不改 jsx-runtime。
3. 不改 docs 和发布策略。

## 完成标准

1. Transition 和 TransitionGroup 不再主动构造 VNode。
2. 进入、离开、移动动画行为与现状一致。
3. 相关测试和示例通过。

## 回退点

如果 TransitionGroup 风险高于 Transition，允许拆成两个提交，但仍保持在同一微阶段主题内。
