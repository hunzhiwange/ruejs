# 11：切换 JSX runtime 与公开类型桥接

## 目标

开始处理用户可见语义，把默认公开类型逐步桥接到 Renderable。

## 只做这些事

1. 调整 FC / render / renderBetween 的类型桥接。
2. jsx-runtime 开始以新协议为默认方向，但仍保留 compat 回退。
3. 保证现有 TSX 用法不突然断裂。

## 建议涉及文件

1. packages/runtime/src/rue.ts
2. packages/jsx-runtime/src/index.ts
3. 必要的类型测试文件

## 不要顺手做

1. 不删 compat。
2. 不改发布策略。
3. 不大面积改 docs。

## 完成标准

1. 公开类型已经能表达 Renderable。
2. 旧 TSX 和手写 h 仍能通过 compat 工作。
3. pnpm check 和相关类型测试通过。

## 回退点

如果公开类型改动影响太大，可先放宽返回类型，再在下一步收紧默认出口。
